# Bayan — Landing Page Plan (v1)

*Last updated: 2026-05-20*

## Page Goal
Primary: Understand problem + solution, share the site
Secondary: Explore interactive demo, read security tips

## Traffic Source
Organic social (Facebook, WhatsApp, Twitter, LinkedIn, TikTok)

## Sections (in order)

### 1. Hero — Stop the Scroll
- Disclaimer banner (top, subtle): هذا مشروع مستقل وليس موقعًا حكوميًا رسميًا
- Headline: الختم راح. الحقيقة ما تروحش.
- Subheadline: بيان يساعدك تتحقّق من البيانات الحكومية في ثوانٍ
- Before/after visual: stamp document → QR document
- CTA: تحقّق الآن ↓

### 2. The Problem
- Title: المشكلة: بيانات مزوّرة في كل مكان
- 3-4 real fake announcement examples
- "Can you tell real from fake?" visual challenge
- Impact cards: stamp forgeable in minutes / no verification exists / wrong decisions daily

### 3. The Solution
- Title: الحل: بيان — تحقّق من كل بيان
- Three components: QR code, verification code, verification page
- Three methods: scan (physical), type (visible code), upload (same phone)
- CTA: جرّب الآن ↓

### 4. Interactive Demo
- Code input field (placeholder: MEN-2026-0847)
- Upload button for screenshots
- Result panel: verified ✅ or not found ❌
- Prompt to try invalid code
- Mock codes: MEN-2026-0847, MES-2026-0132

### 5. Four Pillars
- تحقّق (Verify) — scan, type, upload
- الدليل الرسمي (Directory) — browse all official announcements
- بلّغ (Report) — report suspected fakes
- وعي رقمي (Tips) — digital security awareness

### 6. The Proposal
- Title: مقترح: كيف يمكن للحكومة تبني هذا النظام
- Step-by-step flow: ministry publishes → system generates QR → citizen verifies
- Larger before/after document mockup
- "This is an open proposal" disclaimer

### 7. Digital Security Tips
- Title: وعي رقمي: كيف تحمي نفسك
- 5+ practical tips in cards/accordion
- Real value delivered in v1 regardless of demo

### 8. CTA — Share
- Title: شارك بيان مع من تعرف
- Tagline: إذا ماكانش في بيان، ماشي رسمي
- Share buttons: Facebook, WhatsApp, Telegram, Twitter, Copy Link
- Secondary: follow for updates / v2 roadmap

### 9. Footer
- Bayan logo + tagline
- "مبادرة من yasser.studio"
- Disclaimer (Arabic + French)

## SEO
- Title: بيان — تحقّق من البيانات الحكومية الجزائرية
- Description: منصة مستقلة للتحقق من البيانات الحكومية الجزائرية
- OG image: before/after mockup with Bayan branding

## Technical Checklist
- Full RTL layout
- Mobile-first (tested at 375px)
- Under 2.5s load on 3G
- Arabic alt text on all images
- OG tags with Arabic copy
- WhatsApp/Facebook preview tested
- Disclaimer visible on every viewport
